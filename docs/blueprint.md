# **App Name**: NEULibraryGuard

## Core Features:

- Visitor Authentication & Check-in: Supports visitor entry via RFID card tap (NEU School ID) or institutional Google-based email login. Upon successful authentication, display visitor's name, college/office, and a 'Welcome to NEU Library!' message.
- Purpose of Visit Selection: Visitors must select their purpose of visit from a pre-defined list (e.g., 'reading books', 'research for thesis', 'use of computer', 'doing assignments') during check-in.
- Admin Dashboard: Provides administrators with a comprehensive overview of visitor statistics, including the number of visitors per day, week, month, and customizable date ranges.
- Visitor Access Management: Admins can view all recorded visitor entries and have the capability to temporarily or permanently block specific visitor IDs from future entries via the application.
- PDF Report Generation (tool): A tool allowing administrators to generate and download detailed PDF reports of visitor statistics based on selected date ranges and categories.
- AI Research Resource Suggester (tool): An AI tool that suggests relevant library resources or specific sections of the library based on the visitor's declared 'purpose of visit'.

## Style Guidelines:

- Color palette is designed for a professional, academic, and calm experience, fitting a library environment. The scheme is light. Primary: Deep scholarly indigo (#3E2677) providing a sense of focus and expertise. Background: A very subtle, almost white lavender (#F6F4FB) to maintain visual clarity and support content legibility. Accent: A vibrant electric blue (#446DFE) for interactive elements, highlights, and calls to action, offering a stimulating contrast.
- Headlines and prominent titles will use 'Space Grotesk', a proportional sans-serif with a modern, techy feel suitable for academic and administrative interfaces. Body text, detailed statistics, and reports will utilize 'Inter', a neutral and highly readable sans-serif, ensuring clear communication of information.
- Utilize clean, vector-based line icons with a modern, functional aesthetic. Icons should clearly represent actions (e.g., 'check-in', 'block', 'report') and categories (e.g., 'reading', 'computer use'), ensuring instant recognition and ease of use, particularly for the admin dashboard.
- The layout will be clean, responsive, and data-driven. The visitor check-in interface will be clear and intuitive, emphasizing swift data entry. The admin dashboard will prioritize information hierarchy with clearly defined sections for statistics, visitor lists, and controls, ensuring efficient data review and management across various screen sizes.
- Subtle and purposeful animations will be implemented for user feedback and state changes, such as check-in confirmations, data loading on the dashboard, and transitions between report views, to enhance the user experience without being distracting.
